Truman Course Calendar Extractor Privacy Policy

This Privacy Policy explains how the **Truman Course Calendar Extractor** Chrome extension ("the Extension") collects, uses, and handles your data. The core purpose of this Extension is to help students quickly extract course schedule data from a specific university webpage and automatically add those courses as events to their Google Calendar.

1. Data Collected and How It Is Used

The Extension collects only the minimum data necessary to perform its single stated purpose. **All processing of website content occurs locally within your browser.**

Data Type Data Collected Purpose and Usage
Authentication/Identity Your Google User ID and **Email Address** (via the `identity` permission and Google's OAuth service). Used **solely to authenticate** with the Google Calendar API so the Extension can create and save the course events directly to *your* Google Calendar.
Website Content (Course Data) **Course schedule details** (Course Name, Section Number, Time, Days, and Location) from the current active tab. This data is processed **locally within your browser** (via the `activeTab` and `scripting` permissions) to format the calendar events before sending them to the Google Calendar API.
Local Storage Your **preferred Google Calendar ID** (if selected) and any temporary session tokens. This information is stored **locally in your browser's extension storage** (via the `storage` permission) to maintain your settings and improve user experience. It is **not** transmitted to us.

2. Information Sharing and Disclosure

**We do not sell, rent, trade, or transfer any user data to third parties.**

The data collected is used strictly for the following purpose:

  • **Google Calendar API:** Your **Google OAuth token and course data** are transferred to **Google's servers** solely for the purpose of creating the calendar events you initiate. This action happens under the direct control of the user. We **do not** store your OAuth token or course data on our own servers.

3. Data Retention and Security

  • **Local Data:** Any data stored via the `storage` permission is retained locally on your device until you manually clear your browser data or uninstall the Extension.
  • **Data Security:** The Extension adheres to Google's security practices, including the use of manifest V3 and Google's secure OAuth flow for handling authentication credentials.

4. Compliance with Developer Program Policies

The **Truman Course Calendar Extractor** adheres to the Chrome Web Store Developer Program Policies, including the **Limited Use Policy**. We commit to using your data only for the provision of the core functionality described in this policy.

5. Changes to this Privacy Policy

We may update our Privacy Policy from time to time. We will notify you of any changes by posting the new Privacy Policy on this page.
